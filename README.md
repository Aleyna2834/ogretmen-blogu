# Etkinlik Günlüğü

**Etkinlik Günlüğü**, okul öncesi eğitim alanında etkinlik fikirlerini paylaşmak ve tanıtmak amacıyla hazırlanmış bir web sitesidir. Projede ana sayfa, hakkımda sayfası ve blog yazıları bölümü bulunmaktadır.

Bu proje, okul öncesi öğretmenleri, çocuk gelişimi öğrencileri ve etkinlik fikri arayan kişiler için sade ve anlaşılır bir içerik sunmayı amaçlar.

## Projenin Amacı

Projenin amacı, okul öncesi çocuklara yönelik etkinlikleri düzenli ve görsel olarak desteklenmiş şekilde sunmaktır. Kullanıcılar site üzerinden etkinlik fikirlerini inceleyebilir, öğretici içeriklere ulaşabilir ve okul öncesi eğitim alanında hazırlanmış blog yazılarını okuyabilir.

## Kullanılan Teknolojiler

Projede temel web teknolojileri kullanılmıştır:

* HTML
* CSS
* JavaScript
* Responsive tasarım

## Proje Sayfaları

### Ana Sayfa

Ana sayfa, projenin giriş bölümüdür. Kullanıcıları karşılayan görsel alan, kısa açıklamalar ve site içeriğine yönlendiren bölümler bu sayfada yer alır.

### Hakkımda

Hakkımda sayfasında proje sahibi veya öğretmen hakkında bilgiler bulunur. Bu bölümde eğitim alanı, deneyimler ve okul öncesi etkinliklere yönelik ilgi alanları tanıtılır.

### Blog Yazıları

Blog yazıları sayfasında okul öncesi eğitim, etkinlik fikirleri, oyunlar ve çocuk gelişimiyle ilgili içerikler yer alır. Her blog yazısı kullanıcıların kolayca okuyabileceği şekilde düzenlenmiştir.

## Proje Özellikleri

* Sade ve anlaşılır arayüz
* Ana sayfa, hakkımda ve blog yazıları bölümleri
* Okul öncesi etkinliklere uygun içerik yapısı
* Görsellerle desteklenmiş sayfa tasarımı
* Mobil uyumlu görünüm
* HTML, CSS ve JavaScript ile geliştirilmiş yapı

## Kurulum ve Kullanım

Projeyi çalıştırmak için herhangi bir ek kurulum gerekmez.

1. Proje dosyalarını bilgisayarınıza indirin.
2. `index.html` dosyasını tarayıcıda açın.
3. Ana sayfa, hakkımda ve blog yazıları bölümlerini inceleyin.
